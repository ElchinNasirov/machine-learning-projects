# Sentiment Analysis App

A full-stack web application that analyzes text and predicts whether the sentiment is **Positive** or **Negative**.

## ✨ Features

- Real-time sentiment prediction
- Confidence score with visual progress bar
- Prediction history (last 5 results)
- Clean and responsive user interface
- Flask REST API backend

## 🛠 Tech Stack

- **Frontend**: React
- **Backend**: Flask + Flask-CORS
- **Machine Learning**: RandomForestClassifier (Scikit-Learn)
- **Text Processing**: TF-IDF Vectorizer

## 🚀 How to Run Locally

**Backend:**

cd backend

python3 app.py
##
**Frontend:**

cd frontend

npm start

## 

The React app will open at http://localhost:3000 and connect to the Flask backend at http://127.0.0.1:5000
